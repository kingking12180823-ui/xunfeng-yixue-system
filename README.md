# 巽風易學決策系統 × AI 軍團

本版已將架構調整為：

1. 易學決策系統為主體
2. 客戶基本資料一次輸入
3. 八字、奇門、卜卦／六爻、梅花易數為分析骨架
4. OpenAI、Gemini、DeepSeek 僅作為內部攻防、校核、補強
5. 最後由風羿老師綜合決策引擎輸出正式報告
